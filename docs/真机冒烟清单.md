# 真机冒烟清单(每切片合并前人工过一遍)

> 来源:v3.2 规格 §4。无头测试不可达项:浏览器原生触摸行为、曲面观感、帧率体感。
> 用法:真机打开 Pages 地址(建议无痕窗口避缓存),逐条勾选;不过即记录现象与机型/光线,返工。
> 调试模式:地址后加 `?debug=1` 可看蒙版轮廓/锚点/帧率。

- [ ] **长按 Clear 十连**:四品类各上妆,长按 Clear(手指自然按住即可)连做 10 次,10/10 全卸且有提示,无系统选词/放大镜/菜单弹出
- [ ] **单击 Clear 作用域**:单击只卸当前品类,其余品类妆容原样保留
- [ ] **四品类转头**:四品类全开,左右转头/点头/慢速小幅晃动,眉/眼/颊/唇妆面均无可察觉抖动、无拖影滞后
- [ ] **手涂张嘴**:唇妆亲手上妆涂满,张大嘴说"啊"——上下唇颜色各自跟随,不断裂、不染口腔/牙齿
- [ ] **腮红曲面贴合**:手涂腮红后转头,颜色随脸颊曲面透视变化,无"平面贴片"感;涂不到鼻子、不出脸缘
- [ ] **眼影全覆盖**:手涂可顺滑涂满内眼角→眼尾并到眼尾外延;笔触呈晕染椭圆无"圆点感";一键眼影含眼尾
- [ ] **眉妆贴合**(`?debug=1`):黄色眉妆轮廓线与真实眉毛重合,偏移肉眼不可辨;加粗/挑眉形变自然
- [ ] **拍照**:快门即按即白闪;照片入册画质无可见劣化(JPEG)
- [ ] **帧率体感**:四品类全开+手涂,流畅度不低于上一版本,机身不发烫(`?debug=1` 左上角可读帧率)
- [ ] **布局**:单屏五页签完整可见可点;快门浮钮不遮挡任何控件;大屏(展开)布局正常
