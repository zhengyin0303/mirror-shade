# 镜色 MirrorShade · AR 口红试妆镜

一面诚实的试妆镜:不磨皮、不滤镜,实时还原每一支口红涂在你唇上的样子。
所有画面仅在设备本地处理(MediaPipe 端侧人脸识别),不上传任何影像。

**在线体验**:启用本仓库的 GitHub Pages 后,访问 `https://<你的用户名>.github.io/<仓库名>/`

## 当前版本:v1.3.1(原型阶段)

核心功能:镜面预览 · 开场肤色冷暖分析与显白色推荐(可确认收起/修改)· 一键试色 ·
手指涂抹上妆(锁区/叠涂/卸妆棉)· 唇型控制点编辑(微笑唇/嘟嘟唇/M唇预设)·
哑光/水光质地 · 浓度调节 · 画面缩放(0.9x–3x)· 按住对比素颜 · 拍照保存 · Clear

## 仓库结构

| 路径 | 说明 |
|------|------|
| `index.html` | 应用本体(单文件,无构建依赖) |
| `docs/` | 项目方案文档(定位/竞品/MVP范围/技术方案/路线图) |
| `tests/` | Playwright 端到端自动化测试(25项)+ MediaPipe 人脸模拟器 |

## 本地运行测试套件

需要 Node.js 与 Playwright(含 Chromium):

```bash
node tests/e2e-test.js
```

测试通过网络拦截注入 `tests/mock-facemesh.js`(合成人脸关键点)并使用 Chrome
假摄像头模式,无需真实摄像头即可回归全部交互逻辑。
注意:脚本内的 playwright 引用路径与 HTML 路径需按本机环境调整。
