# 镜色 MirrorShade · AR 试妆镜

一面诚实的试妆镜:不磨皮、不滤镜,实时还原口红与妆容在你脸上的真实样子。
所有画面仅在设备本地处理(MediaPipe 端侧人脸识别),不上传任何影像。

**在线体验**:https://zhengyin0303.github.io/mirror-shade/

## 当前版本:v2.4(二期完成)

功能按界面七个页签组织:

| 页签 | 功能 |
|------|------|
| 试色 | 12 色口红一键上唇,开场肤色冷暖分析(可确认收起/手动修改),★ 标记依肤色推荐 |
| 上妆 | 手指涂抹上妆:大刷/小刷/卸妆棉,锁区防出界,薄涂叠加 |
| 唇型 | 8 控制点唇型编辑,微笑唇/嘟嘟唇/M唇预设,对称开关 |
| 腮红 | 6 色,三种打法:苹果肌/斜扫显瘦/晒伤腮红 |
| 眉妆 | 5 个眉色染色(保留真实眉流),原生/加粗/挑眉三种眉型 |
| 眼影 | 6 色上睑晕染,单色或外眼角渐变 |
| 衣橱 | 口红衣橱:长按收藏/自定义录入/品牌经典色号快填/照片取色;**撞色预警**(Lab ΔE 色差,买前查重);导出/导入,数据仅存本机 |

通用能力:哑光/水光质地、浓度分品类记忆、深唇显色补偿、画面缩放(模糊背景填充)、按住对比素颜、拍照保存、一键全卸妆。

## 本地运行与测试

```bash
npm install          # 首次:安装 Playwright
npm test             # 自动化回归(44 项,无需真实摄像头)
npm run serve        # 本地预览:http://localhost:8000/
```

注意:手机真机测试请使用上方 GitHub Pages 地址——浏览器仅在 https 或 localhost 下放行摄像头,手机访问电脑局域网 IP 会被拒绝。
